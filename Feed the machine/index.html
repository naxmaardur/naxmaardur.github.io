Feed the machine

The theme for this jam was extinction and I decided to design a game round an ever-hungry machine that you build your self but is the cause of the destruction of all the surrounding nature.
This is very clearly meant to represent climate.

For this game because of the scope and the time frame players should be able to experience it I went for an easy-to-understand system, the construction of anything in the game uses 1 or more of 3 different resources: wood, iron, and coal. Using these resources the player can place buildings that gather more of these resources.
The first obstacle players should encounter is that their collection of resources is limited by the amount of robots they have, by constructing a factory they can make more robots but this uses a lot of their resources which they also need to construct the machine in time. This prepares them for the second half of the game where players will need to produce enough robots to keep the machine fed. I also wanted to give players the option to have a moral choice, so a player can choose to "lose" on purpose saving the environment but losing their job.
